## How To Use
clone this repository :
```
git clone https://github.com/robsonmvieira/personal-site
```
 ## Install dependencies
  go to project folder open the terminal and run :
```
  yarn
```
 ## Start the Project
  start the project in development mode :
```
  yarn start
```
